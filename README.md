# Laptop Portfolio Website

A single-page portfolio with a laptop-first concept:

- On load, the UI only shows a laptop.
- Inside the laptop screen, you tap a **folder**.
- A zoom animation "enters" the folder.
- Inside, you can switch between **About**, **Projects**, and **Contact**.

## Files

- `index.html` – app structure and portfolio content.
- `styles.css` – laptop visual design, layout, and animations.
- `script.js` – folder open/close behavior, menu switching, and clock.

## Quick start

Because this project is pure HTML/CSS/JavaScript, you can run it by simply opening `index.html` in your browser.

For best behavior (recommended), serve the folder from a local server using your preferred tool.

## Customize your portfolio

1. Open `index.html`.
2. Replace:
   - `Your Name`
   - project cards
   - email/LinkedIn/GitHub links
3. Save and refresh the page.

## Interaction controls

- Click/tap **Portfolio** folder to open.
- Click **Back to desktop** to close.
- Press **Esc** to close folder quickly.
